Added followng methods to AVL Tree Class:
* [get_max_value](avl.py#L114) - return max value of AVL Tree
* [get_min_value](avl.py#L119) - return min value of AVL Tree
* [get_sum](avl.py#L124) - return sum of AVL Tree elements
